# Code Review Records

이 디렉터리는 개발 단계별 코드 리뷰 결과를 누적 기록한다.

## 우선순위

* `P0`: 데이터 손실, 보안 문제 또는 실행 불능. 즉시 수정한다.
* `P1`: 핵심 기능의 정확성이나 복구 가능성을 훼손할 수 있음. 다음 단계 전에 수정한다.
* `P2`: 특정 입력이나 운영 상황에서 오동작할 수 있음. 가까운 후속 작업에서 수정한다.
* `P3`: 유지보수성, 이식성 또는 운영 안전성을 높이는 개선 사항이다.
